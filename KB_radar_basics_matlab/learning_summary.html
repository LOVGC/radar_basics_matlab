01

核心抽象：一个数据立方体，三种相位

单目标的连续时间回波、矢量形式、数据立方体形式，是同一个物理信号的三种表示， 而不是三个模型。固定其中两个维度，剩下的那个维度的相位规律就对应一个物理参数：

fast time · 距离

脉内采样维。目标距离造成回波延迟 \(\tau = 2R/c\)，由匹配滤波 / 脉冲压缩提取。

$$\Delta R = \frac{c}{2B}$$
slow time · 速度

脉冲间维。径向速度造成逐脉冲相位旋转，由 Doppler FFT 提取。

$$f_D = \frac{2v}{\lambda},\quad \Delta v \propto \frac{1}{T_{\mathrm{CPI}}}$$
array element · 角度

阵元维。波达角造成阵元间相位递进，由波束形成 / 空间 FFT 提取。

$$\Delta\phi = -\frac{2\pi d \sin\theta}{\lambda}$$

统一的处理观：距离、多普勒、角度处理本质上都是匹配滤波 / 模板投影—— 分别用延迟模板、慢时间正弦模板、空间导向矢量去匹配数据，匹配上的 bin 产生相干增益。

02

分辨率与模糊：支撑长度决定一切

三个维度的物理分辨率都由测量支撑（support / aperture）的倒数决定， FFT 补零只是显示插值，不改变物理分辨率：

分辨率 — 由支撑长度决定

  • 距离：带宽 \(B\) 越大峰越窄。10 MHz → ≈15 m；8 m 间隔的两目标合并成一个峰，20 MHz 才能分开（demo_02）。
  • 速度：CPI 时长（脉冲数 \(N_p\)）决定速度 bin 间距。30 与 32 m/s 在 \(N_p=64/128\) 都明显重叠；要 0.5 m/s 的 bin 间距约需 \(N_p \approx 300\)（demo_03）。
  • 角度：阵元数 / 孔径决定波束宽度。\(M=16\) 时角分辨率约 7.84°，\(N_{\mathrm{fft}}\) 从 16 加到 4096 也不变；\(M=32\) 才降到约 3.92°（demo_08）。
  • 名义分辨率 \(\Delta R = c/(2B)\) 是尺度而非硬阈值：略超过它的间隔会部分分开但主瓣仍重叠。

模糊 / 混叠 — 由采样间隔决定

  • PRF 折中：PRF↑ → 最大无模糊距离↓、最大无模糊速度↑。
  • Doppler 混叠：超出无模糊速度区间的目标会折叠，例如速度周期 ≈150 m/s 时，160 m/s 的目标出现在 ≈10 m/s。
  • 空间混叠（栅瓣）：阵元间距是空间采样间隔，\(d \le \lambda/2\) 才无混叠。\(d=\lambda\) 时 30° 目标在 −30° 出现等高栅瓣（0 dB），而 \(d=\lambda/2\) 时该方向约 −308 dB（demo_05）。
  • 时间混叠与空间混叠是同一个采样定理在 slow time 维和 array element 维的两次出现。
03

端到端骨架：从原始立方体到目标参数

demo_06 把三个维度串成一条完整链路，单目标（4 km, 30 m/s, 20°）估计结果为 4002.23 m / 30.45 m/s / 20.00°：

rx[fast, slow, elem] = targetEcho + noise
→ 匹配滤波（沿 fast time，逐 slow-time/通道切片）
→ Doppler FFT（沿 slow time，逐 range/通道切片）
→ 非相干 RD 峰值检出
→ 仅在检出的 RD cell 上做角度扫描（而非对每个 RD cell 全扫）

角度域两种等价视角（demo_07）：常规角度扫描 \(|a(\theta)^H x|\) 与空间 FFT 找的是同一个空间频率 \(u = d\sin\theta/\lambda\)（如 \(d=\lambda/2,\ \theta=30°\) 时 \(u=0.25\)），二者对同一目标分别估出 30.05° 和 30.06°。

04

超分辨与自适应波束：MUSIC 与 MVDR

常规波束形成受孔径限制、也不会主动压制干扰。MUSIC 和 MVDR 用协方差信息换取更强能力， 但代价是更强的模型假设和调参需求。

MUSIC — 子空间超分辨

  • \(M\) 阵元、\(K\) 个源 → 信号子空间维 \(K\)，噪声子空间维 \(M-K\)（快拍数只影响协方差估计质量，不影响子空间维数）。
  • 流程：协方差估计 → 特征分解 → 取噪声子空间 → 伪谱扫描。
  • demo_09：\(M=16, K=2\)，特征值 17.80 / 15.08 / ≈0.03 清晰分层，20° 与 28° 两个近距目标被 MUSIC 分得比常规波束形成尖锐得多。
  • 依赖：已知/估计的源数 \(K\)、足够快拍、非相干源、阵列校准准确。

MVDR — 自适应置零

  • 无失真约束下最小化输出功率：
$$w = \frac{R^{-1}a}{a^H R^{-1} a}$$
  • demo_10：看向 20°、干扰在 −30°，常规波束对干扰增益约 −26.5 dB，MVDR 约 −80.3 dB，且看向增益都保持 0 dB。
  • 对角加载（demo_11）：加载小 → 信任协方差、零陷深（1e−6 → −65 dB）；加载大 → 趋向常规波束、零陷变浅（1e1 → −35 dB）。快拍少 / 协方差不可靠时应加大加载。

关键结论：MUSIC / MVDR 不是"免费升级"——它们需要训练快拍和准确模型，常规波束形成在假设不可靠时反而更稳健。

05

CFAR 检测（当前阶段）

formulation · 检测 = 逐 cell 二元假设检验

检测问题可以 formulate 成：给定 1D/2D/3D 检测统计量 tensor，对每个 cell 做一次 \(H_0\)（只有噪声）vs \(H_1\)（有目标）判决，输出同形状的 0/1 mask tensor。 这个抽象正确，但有两个关键精化：

  • mask 是中间产物，不是最终输出：目标主瓣跨越多个 cell，相邻的 1 须先聚类，再输出带参数估计（距离/速度/角度/SNR）的 detection list。demo_12 中 7 个过门限 CUT → 聚类 → 1 个目标。边缘 cell 凑不齐训练窗，通常无判决。
  • "1" ≠ 有目标，而是"统计量过门限"：噪声 cell 以 \(P_{fa}\) 概率误报 1，弱目标可能漏报 0。mask 是统计判决的一次实现，量化判决器好坏需要 \(P_d\) / \(P_{fa}\) / ROC（即下一阶段内容）。
CA-CFAR · 1D range profile
  • CFAR 在上述 formulation 中的定位：它不定义 mask 问题，而是回答"每个 cell 的门限怎么定"——用 CUT 周围的训练单元自适应估计本地噪声水平，使 \(P_{fa}\) 在不均匀背景下保持恒定（Constant False Alarm Rate 名字的由来）。固定全局门限也能产生 mask，但背景一变 \(P_{fa}\) 就失控。
  • CFAR 是通用方法：不限于 RD 图，可作用于任何检测统计量图/立方体（range profile、RD、RA、DA、RDA tensor），核心都是 CUT + 保护单元 + 训练单元。
  • demo_12 在匹配滤波距离像上实现 1D CA-CFAR：7 个相邻 CUT 过门限 → 聚类成 1 个目标，最强检测在 4002.23 m（真值 4000 m）。
  • 参数直觉（待巩固）：保护单元防目标能量泄入训练区，训练单元越多噪声估计越平滑，\(P_{fa}\) 决定门限缩放因子。

当前薄弱点与下一步

需要巩固

  • 匹配滤波 / Doppler FFT / 波束形成模板中的共轭与符号约定（如 demo_07 对快拍取共轭使峰出现在正 \(u\)）。
  • 区分名义分辨率与"视觉上干净的双峰分离"；区分 Doppler 分辨率与无模糊速度。
  • 角谱中区分真目标峰、旁瓣、噪声峰与栅瓣；均匀加权下增阵元主要变窄主瓣，旁瓣相对电平取决于加窗。
  • 对角加载量与协方差质量、导向失配、稳健性需求之间的实际联系。
  • CFAR 训练/保护单元在 1D、2D、3D 检测产品中的设计差异。
  • 区分 cell 级二元 mask 与聚类后的 detection list：mask 是中间判决结果，目标报告应包含参数估计与检测统计量。
