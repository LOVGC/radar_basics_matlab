同一个信号，两套坐标系

KEY INSIGHTS

最关键的七条

如果只记住几句话，记这些。后面的图都是在把它们讲清楚。

01 · 采样 → 复制

时域采样 = 频域以 Fs 复制

把 \(x_c(t)\) 按 \(F_s\) 采样，频谱会沿频率轴每隔 \(F_s\) 复制一份。DTFT 就是这一堆复制的结果，也因此天然以 \(2\pi\) 为周期。

02 · 对偶

一个域离散 ⟺ 另一个域周期

这一句话同时统一了 CTFT / Fourier Series / DTFT / DFT。哪个域被采样（离散），另一个域就被周期化。

03 · DFT

DFT 是对 DTFT 采样

\(X[k]=X_d(e^{j\Omega})\big|_{\Omega=2\pi k/N}\)。它只取 \(N\) 个等间隔频率样本，并隐含时域周期延拓——这是 spectral leakage 的根源。

04 · 单位

不是 [0,1] Hz，是 [0,1) cyc/sample

同一个点四种写法：f Hz ↔ Ω=2πf/Fs rad/sample ↔ ν=k/N cycles/sample ↔ bin k。

05 · FFT

FFT 不是新变换

它只是把 DFT 从 \(O(N^2)\) 加速到 \(O(N\log N)\) 的算法，数学对象与 DFT 完全相同。

06 · 世界观

换基 = 模板匹配

analysis 用 inner product 量每个频率成分（匹配模板），synthesis 把成分加回去重构信号（换基）。正交基让两者是同一件事。

07 · 为什么是它

Fourier 基是 LTI 的特征基

复指数是 LTI 系统的 eigenfunction，所以 Fourier 把卷积对角化成逐点相乘。这才是它在信号处理里独一无二的真正原因。

PART 01

从 CTFT 到 FFT：一条采样链

这四个名字其实是沿着一条链往下走：先在时域采样，再在频域采样，最后用一个算法把它算快。每一步都在两个域之间制造"离散↔周期"的对偶。

全景：三个域，两次采样 总览

从左到右是三个落脚点。每个面板上方是时域信号，下方是频谱。注意频谱形状随每一步采样如何"长出周期"。

① 采样为什么产生频谱复制 CTFT → DTFT

时域采样等价于乘上一串冲激（impulse train），而时域相乘 = 频域卷积。冲激串的频谱还是一串冲激（间隔 \(F_s\)），所以原谱被卷积后就沿频率轴每隔 \(F_s\) 复制一份。映射关系是 \(\Omega = 2\pi f / F_s\)。

\[ X_d(e^{j\Omega}) = F_s\sum_{k=-\infty}^{\infty} X_c\!\Big(F_s\big(\tfrac{\Omega}{2\pi}-k\big)\Big) \]

关键后果：如果信号最高频率 \(B\) 满足 \(F_s < 2B\)（违反 Nyquist），相邻复制就会重叠 = 混叠 aliasing。拖动滑块看这件事发生：

② 四种变换的统一：离散 ⟺ 周期 对偶

把"时间是否离散"和"信号是否周期"做成 2×2，四种 Fourier 工具刚好各占一格。规律只有一条：一个域被离散化，对偶域就被周期化。

③ DFT = 在一个周期上取 N 个频率样本 DTFT → DFT

长度 \(N\) 的序列，其 DFT 正是 DTFT 在 \(\Omega_k = 2\pi k/N\) 处的采样：

\[ X[k]=\sum_{n=0}^{N-1}x[n]\,e^{-j2\pi kn/N}=X_d(e^{j\Omega})\Big|_{\Omega=2\pi k/N},\quad k=0,\dots,N-1 \]

因为在频域采样，对偶地时域被周期延拓：DFT 默认你这 N 个点是某个周期信号的一个周期。频率分辨率（bin 间距）由此固定为 \(F_s/N\)，spectral leakage、windowing 这些现象都从这里来。

④ 把频率单位一次对齐 units

对话里那句"DFT 把频率 normalize 到 [0,1] Hz"需要修正：是 [0,1) cycles/sample，与采样率无关；只有再乘上 \(F_s\) 才回到物理 Hz。拖动滑块，看同一个点在四条轴上的四种读数同步移动：

⑤ FFT：只是把同一个 DFT 算快 algorithm

FFT 利用复指数的对称性做分治，把 \(O(N^2)\) 降到 \(O(N\log N)\)，但输出和直接算 DFT 一模一样。它是算法，不是新的变换对象。

一句话串起来

CTFT 是连续信号的频谱；采样后频谱按 Fs 周期复制 → DTFT；在一个周期上取 N 个频率样本 → DFT（同时隐含时域周期化）；FFT 是快速计算 DFT 的算法。

PART 02

Fourier 的世界观

换基、模板匹配、LTI 特征基——这是同一件事被越看越深的三层。把它们看穿，Fourier 就从"公式"变成了"投影"。

① 换基：同一个向量，换坐标系 change of basis

长度 \(N\) 的信号是 \(\mathbb{C}^N\) 里的一个向量。DFT 把它从 standard basis \(\{e_n\}\) 转到 Fourier basis \(\{\varphi_k[n]=e^{j2\pi kn/N}\}\)。\(X[k]\) 就是它在第 \(k\) 个 Fourier 方向上的坐标——没有创造信息，只是换了一套轴。

② 模板匹配：每个系数都是一次内积 template matching

把 \(X[k]=\langle x,\varphi_k\rangle\) 读成：拿信号去和频率 \(k/N\) 的复正弦做内积，内积大 = 信号里这个频率成分多。下面把一个由两个正弦合成的信号，用一支可调频率的"探针模板"去扫——你会看到匹配强度在信号真正含有的频率处冲高。这个扫描出来的曲线，就是频谱本身。

为什么这两件事是同一件事

当基 \(\{\varphi_k\}\) 正交时，向量在某方向上的坐标 恰好等于 它与该基向量的内积：\(c_k=\langle x,\varphi_k\rangle\)。所以 analysis（用内积找成分强度）和 synthesis（用成分重构）只是一枚硬币的两面。

③ "任何信号都能被这组基表示吗？"——要加限定 completeness

这句话在不同变换里有不同的精确版本，核心差别在于基是可数离散还是连续，以及空间是否 complete。

DFT
finite · clean

完全成立。\(\mathbb{C}^N\) 里任意向量都能由 \(N\) 个 DFT 基精确表示。最有限、最干净的线代情形。

Fourier Series
periodic · countable

对周期信号，基是可数的离散谐波 \(\{e^{jk\omega_0 t}\}\)。适合周期信号的展开。

CTFT
aperiodic · generalized

频率连续，\(e^{j\omega t}\) 能量无限、不属于普通 \(L^2\)，是 generalized basis，要用 distribution / Hilbert 空间语言才严谨。

④ 真正的原因：Fourier 基是 LTI 的特征基 eigenbasis

复指数过 LTI 系统不改变形状，只被缩放和相移——它是 LTI 系统的 eigenfunction，\(H(\omega)\) 是对应的特征值。于是时域里麻烦的卷积，在频域变成简单的逐点相乘：Fourier 把卷积算子对角化了。这才是它区别于"随便一组正交基"的地方。

\[ e^{j\omega t}\;\xrightarrow{\;\text{LTI}\;}\;H(\omega)\,e^{j\omega t}, \qquad y=h*x\;\Longleftrightarrow\;Y(\omega)=H(\omega)\,X(\omega) \]

⑤ 同一个抽象，推广到一切变换 generalization

一旦把它看成"用一组 probes/templates 去测量信号"，很多方法都落进同一个框架。Fourier 的特别只在于它的模板是复指数——也就是 shift / 卷积 / LTI 的天然语言。

transform = analysis operator：用一组模板测量信号，得到系数。inverse transform = synthesis operator：用这些系数把信号重建。当模板张成的空间 complete 时，重建是无损的。Fourier 只是这个范式里，模板恰好是复指数、因而能对角化一切 shift-invariant 现象的那个特例。
