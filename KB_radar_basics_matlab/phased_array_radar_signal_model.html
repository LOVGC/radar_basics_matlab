01

阵列几何 array geometry

考虑一个 M_x × M_y 的 uniform rectangular array(URA)。第 (m, n) 个阵元的位置为:

$$\mathbf{r}_{m,n}=\begin{bmatrix} m\,d_x \\ n\,d_y \\ 0 \end{bmatrix},\qquad m=0,\dots,M_x-1,\quad n=0,\dots,M_y-1.$$

目标方向由 azimuth θ 与 elevation φ 描述,对应的方向单位向量为:

$$\mathbf{u}(\theta,\phi)=\begin{bmatrix} \textcolor{#4CC4FF}{\cos\phi}\,\textcolor{#FF6FA5}{\cos\theta} \\[2pt] \textcolor{#4CC4FF}{\cos\phi}\,\textcolor{#FF6FA5}{\sin\theta} \\[2pt] \textcolor{#4CC4FF}{\sin\phi} \end{bmatrix}.$$
  • θ = 0 目标在 +x 方向的竖直平面内。
  • θ = 90° 目标在 +y 方向的竖直平面内。
  • φ = 0 目标在阵列平面方向,即 horizon。
  • φ = 90° 目标正对 boresight,即 +z 方向。

上方交互图就是这套几何:θ 是 x–y 平面里的粉色弧,φ 是抬起的青色弧,紫色箭头是 u。

02

方向余弦 direction cosines

定义两个 direction cosines —— 它们正是 u 在阵列平面上的两个投影分量:

$$\xi=\textcolor{#4CC4FF}{\cos\phi}\,\textcolor{#FF6FA5}{\cos\theta},\qquad \eta=\textcolor{#4CC4FF}{\cos\phi}\,\textcolor{#FF6FA5}{\sin\theta}.$$

因为阵列只分布在 x–y 平面,空间相位只取决于 u 的平面投影。第 (m, n) 个阵元相对参考阵元的 path difference 为:

$$\mathbf{r}_{m,n}^{\mathsf T}\mathbf{u}=m\,d_x\,\textcolor{#4CC4FF}{\cos\phi}\textcolor{#FF6FA5}{\cos\theta}+n\,d_y\,\textcolor{#4CC4FF}{\cos\phi}\textcolor{#FF6FA5}{\sin\theta}=m\,d_x\,\xi+n\,d_y\,\eta.$$

因此 steering response(单个阵元的相位)为:

$$a_{m,n}(\theta,\phi)=\exp\!\left[\,j\frac{2\pi}{\lambda}\big(m\,d_x\,\xi+n\,d_y\,\eta\big)\right].$$
03

二维 steering vector Kronecker product

因为相位在 x、y 两个方向上是各自线性叠加的,沿 x、沿 y 的 steering vector 分别为:

$$\mathbf{a}_x=\begin{bmatrix}1\\ e^{j\frac{2\pi}{\lambda}d_x\xi}\\ \vdots\\ e^{j\frac{2\pi}{\lambda}(M_x-1)d_x\xi}\end{bmatrix},\qquad \mathbf{a}_y=\begin{bmatrix}1\\ e^{j\frac{2\pi}{\lambda}d_y\eta}\\ \vdots\\ e^{j\frac{2\pi}{\lambda}(M_y-1)d_y\eta}\end{bmatrix}.$$

整个 URA 的 steering vector 因此可写成 Kronecker product:

$$\mathbf{a}(\theta,\phi)=\mathbf{a}_y(\theta,\phi)\otimes\mathbf{a}_x(\theta,\phi).$$

stack 顺序取决于如何把二维阵列 reshape 成一维向量:先 stack x 再 stack y 就用这个形式;顺序反过来,Kronecker 的顺序也要跟着反。

04

单目标接收信号 continuous-time

雷达发射 baseband waveform s(t),载频 f_c,波长 \(\lambda = c/f_c\)。一个目标的参数是 \((R, v, \theta, \phi, \alpha)\),其中:

$$\tau=\frac{2R}{c}\ \text{(round-trip delay)},\qquad f_D=\frac{2v}{\lambda}\ \text{(Doppler)},\qquad t_p=p\,\PRI.$$

那么第 (m, n) 个阵元、第 p 个 pulse 上的 baseband echo 为:

$$x_{m,n,p}(t)=\alpha\,\textcolor{#2FE0C0}{s(t-\tau)}\;\textcolor{#F4B740}{e^{\,j2\pi f_D\,p\PRI}}\;\textcolor{#B78CFF}{e^{\,j\frac{2\pi}{\lambda}\left(m d_x\cos\phi\cos\theta+n d_y\cos\phi\sin\theta\right)}}+w_{m,n,p}(t).$$

这就是最核心的二维相控阵模型 —— 一个目标在数据里同时留下三类相位结构:

s(t − τ)→ range 距离 e^( j2π f_D p T )→ Doppler 速度 e^( j·φ(m,n) )→ 2D angle 角度
05

多目标模型 multi-target

多目标只是线性叠加。对 Q 个目标:

$$x_{m,n,p}(t)=\sum_{q=1}^{Q}\alpha_q\,\textcolor{#2FE0C0}{s(t-\tau_q)}\;\textcolor{#F4B740}{e^{j2\pi f_{D,q}p\PRI}}\;\textcolor{#B78CFF}{e^{j\frac{2\pi}{\lambda}\left(m d_x\cos\phi_q\cos\theta_q+n d_y\cos\phi_q\sin\theta_q\right)}}+w_{m,n,p}(t),$$
$$\tau_q=\frac{2R_q}{c},\qquad f_{D,q}=\frac{2v_q}{\lambda},\qquad \xi_q=\cos\phi_q\cos\theta_q,\qquad \eta_q=\cos\phi_q\sin\theta_q.$$
06

匹配滤波后的离散模型 matched filter + range sampling

经过 matched filtering 与 range sampling,令 range bin index 为 k:

$$x_{m,n,p,k}=\sum_{q=1}^{Q}\alpha_q\,\textcolor{#2FE0C0}{h[k-k_q]}\;\textcolor{#F4B740}{e^{j2\pi f_{D,q}p\PRI}}\;\textcolor{#B78CFF}{e^{j\frac{2\pi}{\lambda}\left(m d_x\cos\phi_q\cos\theta_q+n d_y\cos\phi_q\sin\theta_q\right)}}+w_{m,n,p,k}.$$

这里 h[k − k_q] 是 range matched-filter response,目标对应的 range bin 为:

$$k_q\approx\frac{\tau_q}{T_s}=\frac{2R_q}{c\,T_s}.$$

如果目标刚好落在某个 bin,可简化为(\(\mathcal{Q}_k\) 是落在第 k 个 bin 的目标集合):

$$x_{m,n,p,k}=\sum_{q\in\mathcal{Q}_k}\alpha_q\,\textcolor{#F4B740}{e^{j2\pi f_{D,q}p\PRI}}\;\textcolor{#B78CFF}{e^{j\frac{2\pi}{\lambda}(\cdots)}}+w_{m,n,p,k}.$$
07

向量形式 stack the array

把所有阵元堆成一个向量 \(\mathbf{x}_{p,k}\in\mathbb{C}^{M_xM_y}\),模型变成 steering vector 的加权和:

$$\mathbf{x}_{p,k}=\sum_{q=1}^{Q}\alpha_q\,\textcolor{#2FE0C0}{h[k-k_q]}\;\textcolor{#F4B740}{e^{j2\pi f_{D,q}p\PRI}}\;\textcolor{#B78CFF}{\mathbf{a}(\theta_q,\phi_q)}+\mathbf{w}_{p,k},$$

其中 \(\textcolor{#B78CFF}{\mathbf{a}(\theta_q,\phi_q)}\) 就是第 03 节那个二维 URA steering vector。

08

空时模型 space-time, rank-1

再定义 Doppler steering vector(沿脉冲方向):

$$\mathbf{b}(f_D)=\begin{bmatrix}1\\ e^{j2\pi f_D\PRI}\\ \vdots\\ e^{j2\pi f_D(P-1)\PRI}\end{bmatrix}.$$

把所有 pulses 也堆起来,某个 range bin 内的数据矩阵为:

$$\mathbf{X}_k=\sum_{q=1}^{Q}\alpha_q\,\textcolor{#2FE0C0}{h[k-k_q]}\;\textcolor{#B78CFF}{\mathbf{a}(\theta_q,\phi_q)}\,\textcolor{#F4B740}{\mathbf{b}^{\mathsf T}(f_{D,q})}+\mathbf{W}_k.$$

所以每个 point target 在一个 range bin 里,就对应一个 rank-1 的空时分量 —— 角度 steering vector 与 Doppler steering vector 的外积:

$$\textcolor{#B78CFF}{\mathbf{a}(\theta_q,\phi_q)}\;\textcolor{#F4B740}{\mathbf{b}^{\mathsf T}(f_{D,q})}.$$
09

加入发射波束赋形 transmit beamforming

若发射端也用二维相控阵,并采用发射 beamforming 向量 \(\mathbf{w}_T\),则目标方向上的发射增益为:

$$G_T(\theta,\phi)=\mathbf{w}_T^{\mathsf H}\,\mathbf{a}_T(\theta,\phi).$$

接收模型只是给每个目标再乘上这个发射增益:

$$\mathbf{x}_{p,k}=\sum_{q=1}^{Q}\alpha_q\,G_T(\theta_q,\phi_q)\,\textcolor{#2FE0C0}{h[k-k_q]}\;\textcolor{#F4B740}{e^{j2\pi f_{D,q}p\PRI}}\;\textcolor{#B78CFF}{\mathbf{a}_R(\theta_q,\phi_q)}+\mathbf{w}_{p,k}.$$

这里 \(\mathbf{a}_T\) 是 transmit steering vector,\(\mathbf{a}_R\) 是 receive steering vector。若收发共用同一个 URA,则通常:

$$\mathbf{a}_T(\theta,\phi)=\mathbf{a}_R(\theta,\phi)=\mathbf{a}(\theta,\phi).$$
10

张量数据立方体 tensor data cube

完整的 radar data cube 是一个四维张量 x[m, n, p, k]:

$$x[m,n,p,k]=\sum_{q=1}^{Q}\alpha_q\,\textcolor{#2FE0C0}{h[k-k_q]}\;\textcolor{#F4B740}{e^{j2\pi f_{D,q}p\PRI}}\;\textcolor{#B78CFF}{e^{j\frac{2\pi}{\lambda}\left(m d_x\cos\phi_q\cos\theta_q+n d_y\cos\phi_q\sin\theta_q\right)}}+w[m,n,p,k].$$
  • m x-array index
  • n y-array index
  • p pulse / slow-time index
  • k range-bin index

要估计的目标参数是 \((R_q,\ v_q,\ \textcolor{#FF6FA5}{\theta_q},\ \textcolor{#4CC4FF}{\phi_q},\ \alpha_q)\) —— 即 range、radial velocity、azimuth、elevation、complex reflectivity。

一个很重要的直觉:URA 本质上测的是两个 spatial frequencies:

$$\omega_x=\frac{2\pi d_x}{\lambda}\cos\phi\cos\theta,\qquad \omega_y=\frac{2\pi d_y}{\lambda}\cos\phi\sin\theta.$$

从它们就能反推 azimuth / elevation:

$$\textcolor{#FF6FA5}{\theta}=\operatorname{atan2}\!\big(\omega_y/d_y,\ \omega_x/d_x\big),\qquad \textcolor{#4CC4FF}{\cos\phi}=\sqrt{\left(\frac{\lambda\,\omega_x}{2\pi d_x}\right)^{2}+\left(\frac{\lambda\,\omega_y}{2\pi d_y}\right)^{2}}.$$

所以二维相控阵,本质上是在估计目标方向向量在阵列平面上的投影。

一句话总结:一个点目标 = (range bin 上的 h[k−k_q]) × (脉冲方向的 Doppler 相位) × (阵元平面上的 二维线性相位) —— 在空时上是 rank-1,在角域上测两个空间频率反解方位/俯仰。
约定:θ = azimuth(方位角),φ = elevation(俯仰角);阵列在 x–y 平面,boresight 指向 +z。示意图与公式中的配色一一对应。
